## What changed

### Shader — `src/shaders/ChromatophoreMaterial.js` rewritten twice
- v2: 3 named layers (leukophore / iridophore / chromatophore), dorsal zebra band, global burst() + propagationWave().
- v3 (after critic): iridophore hue centre shifted to cyan so palette sweeps cyan→gold→magenta (was stuck warm); zebra sharpened via `pow(stripe, 6)` with domain-warp wobble; chromatophore density dropped so leukophore base shows through between sacs; global burst replaced with **per-fragment `localBurst()`** — a Poisson-ish scatter of burst centres each with their own start time and `exp(-age*3.5)` decay envelope, so regions burst independently instead of the whole body pulsing in sync; added `dorsalWave()` that travels front→back along the mantle.

### Mantle UV mapping — `src/entities/Cuttlefish.js buildMantle()`
- u = (phi + π/2) / 2π → wraps circumference at constant density; 0.5 = dorsal top.
- v = t → runs head→tail.
- Kills the stretched-chromatophore artefact the user flagged.

### Geometry — flat shield + radial arm crown
- Mantle defaults: radius 0.42 (wider), height 0.22 (flatter) → shield not loaf.
- `armSpine` rewritten as radial crown — 8 arms evenly spaced around the mouth axis at distance `armCrownRadius=0.09`, each with per-pair length scale `[0.85,0.95,1.00,1.10]`.
- Tentacles longer (1.8), thinner base (0.018), bigger club (0.065), club bump starts at t>0.72, retracted path tucks inside the arm crown.
- Side fins: width 0.16, mask floor 0.4, outer edge curves downward, attaches slightly below midline (finAttachY -0.03).

### Animation — `updateCuttlefish()`
- Arms: per-vertex travelling sine wave along each tentacle spine, amplitude grows with u (tips whip, bases anchored).
- Tentacles: per-segment phase offset with u² amplitude scaling.
- Mantle `uTime` pumped every frame via set-of-materials so material reuse works.
- Verified via 6-frame animation strip at 300 ms intervals (`cuttle_anim_v005.png`) — chromatophore patches are visibly different between frames.

### Cuttlefish eye
- Real W-pupil opaque on top of iris (was hidden behind translucent cornea before).
- Eyes sunk into mantle via `eyeSocketDepth` offset along normal.

### Fish eye
- Hemisphere pupil now recessed `-r*0.18` for stronger parallax.
- Glint pushed forward to `z = r*0.55` so parallax is visible at oblique angles.
- Iris cooled to teal/grey (`0x3a4a5a`) — no more red.
- Partial-arc shimmer ring (lower 60% only) instead of full halo.
- Socket ring adds depth cue that the eye is inset.
- Radius clamp softened: `max(0.018, min(r, 0.055))` — no more pinprick eyes on big fish.

### Per-species iridophore — every fish now has it
- `FishManager.addFish()` reads iridophore multiplier from species JSON, falls back to pattern-type lookup (showy two-tone stripes = 0.55, mottled bottom-dweller = 0.10).
- Spectral bias randomised per fish from UUID byte → each fish has a different thin-film hue.
- `uTime` pumped per fish every frame.

### Variant sheet — `cuttlefish-sheet.html` / CuttlefishSheet.js
- Rows now couple `(mantleLength, mantleRadius, mantleHeight, mantleTaper)` → stout / standard / elongated / eel-like, visibly different silhouettes.
- Columns couple `(armCurl, tentacleExtension, finRippleAmp, zebraIntensity, chromaIntensity)` → calm / drifting / alert / striking, clear posture progression.

### Skin texture sheet — new `skin-sheet.html`
- 16 flat UV-plane tiles with live shader. Rows = zebra settings, cols = iridophore × density.
- Master reference image embedded top-left for side-by-side comparison.

### Tooling
- `agents/tools/animation_strip.mjs` — captures N frames at T spacing, composes a horizontal strip with timestamps.
- `agents/tools/debug_page.mjs` — captures all page console + errors (caught the "backtick-in-GLSL-comment" JS parse error).
- `agents/tools/screenshot_url.mjs`, `compare_sheet.mjs` retained.
