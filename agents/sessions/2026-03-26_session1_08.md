## What Was Built

### Agent Infrastructure (`agents/`)
- `AGENTS.md` — System architecture doc with roles, review cycle, progress tracking
- `PLAN.md` — Master plan with phases, checkboxes, progress log
- `roles/` — 5 agent role definitions:
  - Geometry Expert — fish morphology and spline curves
  - Texture Expert — procedural texture parameter exploration
  - Color Expert — species-accurate coloration
  - Critic — adversarial quality review (scoring 1-10, REJECT/REVISE/ACCEPT)
  - Assembler — combines approved components
- `progress/reviews/` — Formal critic reviews with scores

### Rendering Tools (`agents/tools/`)
- `render_fish_sprite.js` — Headless fish-to-PNG renderer
  - Catmull-Rom cubic spline interpolation for body outlines
  - Separate fin structures (dorsal, caudal, anal, pectoral)
  - Species profiles with 16 control points each (v2)
  - `--all` mode generates all species + contact sheet
  - `--mode silhouette|textured|colored`
  - `--debug` (planned) for control point toggle
- `render_texture_tile.js` — Texture parameter explorer
  - Scales (ridge noise + voronoi), Spots (voronoi), Stripes (fbm wobble)
  - `--sweep` mode generates parameter grid
  - `--batch` mode for bulk generation
  - Gradient noise, ridge noise, voronoi, fbm primitives
- `render_contact_sheet.js` — Grid assembler for comparison
