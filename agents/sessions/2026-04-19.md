# Session log 2026-04-19

## What changed
- **Launcher scripts** — `scripts/launch-aquarium.sh`, `launch-workshop.sh`, `launch-spinner.sh`, `launch-cuttlefish.sh`, `ensure-dev.sh`, `dev.sh`.
- **Examples reorg** — all third-party reference images moved into `examples/external/`; `examples/internal/` holds our own screenshots and iteration sheets; `examples/reference/` reserved for curated favourites.
- **Plan updated** — `plan.md` now tracks three iteration tracks (A: fish eyes, B: iridophore layer, C: cuttlefish) + tooling infra.
- **Fish eyes upgraded** in both `src/entities/Fish3DBuilder.js` (main tank) and `src/entities/FishGeometry.js` (designer preview) from 3 flat circles → 7-layer shimmer eye: sclera + gold iris ring + pupil + additive-blend rainbow shimmer ring + glossy cornea hemisphere + main glint + cyan sparkle. FishManager spins the shimmer ring per frame.
- **Iridophore thin-film layer** added as Layer 3b in `src/shaders/FishShaderMaterial.js`. Approximates thin-film interference as `spectrum(pathLen * cosθ + noise)` with view-angle falloff. Spinner page has 5 new sliders (`iridoIntensity`, `iridoThickness`, `iridoSpectralBias`, `iridoMaskScale`, `iridoMaskOpacity`). Tank fish ship with `iridoIntensity=0.25` for subtle shimmer.
- **Chromatophore shader extracted** — `src/shaders/ChromatophoreMaterial.js` is a reusable port of the workshop shader (voronoi sacs, rotating iridescence, leucophore base) that takes a world-space normal so it lights correctly on a 3D surface.
- **Cuttlefish 3D geometry** — new `src/entities/Cuttlefish.js` builds mantle (tapered ovoid), 2 side fins (rippling ribbon), 8 arms (Catmull-Rom tapered tubes), 2 hunting tentacles with clubbed tips, W-pupil eyes. `updateCuttlefish(group, t)` animates fin ripple + arm sway + tentacle idle + pushes time into the chromatophore shader.
- **Cuttlefish preview page** — `cuttlefish-preview.html` with live sliders for every param.
- **Variant sheet page** — `cuttlefish-sheet.html` renders a live 4×4 grid sweeping mantleTaper × armCurl.
- **In-tank cuttlefish** — `src/entities/CuttlefishCreature.js` spawns one cuttlefish that drifts, occasionally darts, respects tank bounds. Hooked into `src/core/Scene.js`.
