Aquarium repo operating notes:

- The publish standard lives in `agents/KB.md`.
- Use the local ignored `.env` for DreamHost deploy settings.
- `claudeaquarium` is the current DreamHost user for this repo.
- Run `./scripts/setup-dreamhost-ssh.sh` once to install the key. It makes exactly one password SSH attempt.
- After setup, deploys are key-only:
  - `./scripts/publish-aquarium.sh`
  - `./scripts/publish-cuttlefish.sh`
- Standard aquarium should stay free of the removed holo tetra squad / holo fish layer.
